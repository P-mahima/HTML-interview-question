1.Difference between the div and span.

div is a block level element. And it takes whole width of the webpage
span is a inline element. And it takes the width as per the element present init.

2.Difference between the class and id.

id is must be unique.
class need not be unique.

3.What are the self closing tag?

self closing tag are those tag that they do not need any closing tag.
eg. img, hr, br, input, meta

4.How many types of headings are there?

there are six types of headings.
  • h1 = 32px

5.Difference between block and inline.

block elements takes whole width of the webpage. and they always start on the new line.
eg. div, p, h1-h6, table, li.
inline elements take the width as per the content present inside it. it just start one another, doesn't start one another.
eg. img, a, span, input, button, iframe.

6..Difference between em and strong.

Both are the inline element.
em is emphasis tag which is used to make the content itallic.
strong tag is used when we have to give importance to some specific content/text.

7.What are the attributes?

Attributes are the property of the tag.
eg.
img tag has the attributes named as src="" and alt="".
a (anchor) tag has the attributes named as href="" and target="".
